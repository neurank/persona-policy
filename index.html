Политика обработки персональных данных

  1. Общие положения
    • Настоящая Политика обработки персональных данных (далее — «Политика») разработана в соответствии с Федеральным законом от 27.07.2006 №152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных пользователей сервиса Persona (далее — «Сервис»).
    • Оператором персональных данных является владелец Сервиса Persona (далее — «Оператор»).
    • Использование Сервиса и нажатие кнопки «Начать» означает выражение пользователем согласия на обработку персональных данных в соответствии с настоящей Политикой.
    • Сервис не является медицинской, психологической или психотерапевтической консультацией.

  3. Состав обрабатываемых персональных данных

    В рамках использования Сервиса могут обрабатываться следующие персональные данные:

    • Telegram ID пользователя
    • имя пользователя (username), при наличии
    • пол
    • возрастная категория
    • ответы, выбранные в анкете
    • текстовые ответы, введённые пользователем
    • технические данные взаимодействия с ботом (дата, время, этапы прохождения)

    Сервис не запрашивает специальные категории персональных данных (о здоровье, религии, политических взглядах и т.д.).

    В случае добровольного предоставления таких сведений пользователем в текстовых ответах они обрабатываются исключительно в рамках целей, указанных в настоящей Политике.

  4. Цели обработки персональных данных

    Персональные данные обрабатываются в следующих целях:

    • Формирование персонализированного анализа на основе ответов пользователя
    • Автоматическая генерация сравнительного разбора возможных стратегий и сценариев
    • Обеспечение функционирования Сервиса
    • Улучшение качества работы Сервиса
    • Проведение обезличенного статистического анализа
  5. Правовые основания обработки

    Обработка персональных данных осуществляется на основании:

    • статьи 6 Федерального закона №152-ФЗ
    • согласия субъекта персональных данных, выраженного путём совершения конклюдентных действий (использование Сервиса и нажатие кнопки «Начать»)
  6. Принципы обработки персональных данных

    Оператор осуществляет обработку персональных данных на основе следующих принципов:

    • законность и справедливая основа обработки
    • ограничение обработки достижением конкретных, заранее определённых целей
    • недопущение избыточности обрабатываемых данных
    • обеспечение точности данных
    • хранение не дольше, чем этого требуют цели обработки
  7. Характер обработки
    • Обработка персональных данных осуществляется преимущественно в автоматизированной форме.
    • Формирование анализа осуществляется цифровой системой без участия человека.
    • Сервис не принимает решений, порождающих юридические последствия для пользователя.
    • Персональные данные не передаются третьим лицам в маркетинговых целях.
    • Для технической обработки могут использоваться сторонние сервисы обработки данных, действующие на основании договоров и обязанные соблюдать требования законодательства РФ о персональных данных.
  8. Обезличивание данных

    Персональные данные могут быть обезличены и использованы в агрегированном виде для:

    • анализа повторяющихся пользовательских сценариев
    • улучшения алгоритмов работы Сервиса
    • статистических целей

    Обезличенные данные не позволяют идентифицировать конкретного пользователя.

  9. Трансграничная передача данных

    В случае использования сторонних вычислительных сервисов возможна трансграничная передача персональных данных.

    Такая передача осуществляется в соответствии со статьёй 12 Федерального закона №152-ФЗ при условии обеспечения надлежащей защиты прав субъектов персональных данных.

  10. Срок хранения персональных данных

    Персональные данные хранятся не дольше, чем это необходимо для достижения целей обработки.

    Удаление персональных данных осуществляется:

    • по запросу пользователя
    • при отзыве согласия
    • по истечении срока хранения
    • при прекращении работы Сервиса
  11. Права субъекта персональных данных

    Пользователь имеет право:

    • получать информацию о своих персональных данных
    • требовать их уточнения, блокирования или уничтожения
    • отозвать согласие на обработку
    • обжаловать действия Оператора в уполномоченный орган по защите прав субъектов персональных данных

    Для реализации прав пользователь может обратиться к Оператору по контактам, указанным в Сервисе.

  12. Меры по обеспечению безопасности

    Оператор принимает необходимые правовые, организационные и технические меры для защиты персональных данных от:

    • неправомерного или случайного доступа
    • уничтожения
    • изменения
    • блокирования
    • копирования
    • распространения
  13. Изменение политики
    Оператор вправе вносить изменения в настоящую Политику. Актуальная версия всегда доступна по адресу: https://neurank.github.io/persona-policy/
